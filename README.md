# Mini Crane Project

This repository contains the documentation and resources for my Mini Crane project, completed as part of my Individual Technical Project for professional career preparation. The project demonstrates a crane system controlled by an STM32 Nucleo microcontroller, with sensors and manual controls for crane motion and hook manipulation.
